# Features
Can render the seconds left using ASCII table and console only. 
```
cd /RenderTimeOut
dotnet run [seconds]
```
Timer has 4 states: 
1. Pause
2. Resume
3. Stop
4. Reset

## Example 
```dotnet run 10```
